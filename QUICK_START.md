# Быстрый запуск приложения

## Шаг 1: Запустить MySQL

1. Откройте **XAMPP Control Panel**
2. Нажмите **Start** рядом с **MySQL**
3. Дождитесь, пока статус станет зелёным

## Шаг 2: Создать базу данных (если ещё не создана)

### Вариант А: Через веб-интерфейс (рекомендуется)

1. Запустите PHP сервер (см. Шаг 3)
2. Откройте в браузере: **http://localhost:8000/setup_database.php**
3. Введите данные подключения к MySQL:
   - Хост: `localhost`
   - Пользователь: `root` (или другой, если создали)
   - Пароль: (введите пароль или оставьте пустым)
4. Нажмите "Создать базу данных"
5. База данных и все таблицы будут созданы автоматически

### Вариант Б: Через phpMyAdmin

1. Откройте **phpMyAdmin**: http://localhost/phpmyadmin
2. Импортируйте файл `database/database.sql`
   - Выберите базу данных (или создайте новую `ритуальные_услуги`)
   - Перейдите на вкладку "Импорт"
   - Выберите файл `database/database.sql`
   - Нажмите "Вперёд"

## Шаг 3: (Опционально) Заполнить БД тестовыми данными

1. Откройте: **http://localhost:8000/scripts/fill_database.php**
2. Нажмите "Заполнить базу данных"
3. Дождитесь завершения процесса

## Шаг 4: Запустить PHP сервер

### Вариант А: Через батник (проще всего)
1. Дважды кликните на файл `bat\start.bat`
2. Оставьте окно открытым

### Вариант Б: Через командную строку
1. Откройте командную строку в папке проекта
2. Выполните:
   ```bash
   C:\xampp\php\php.exe -S localhost:8000
   ```

### Вариант В: Через XAMPP Apache
1. В XAMPP Control Panel нажмите **Start** рядом с **Apache**
2. Откройте в браузере: `http://localhost/DBGranit`

## Шаг 5: Открыть в браузере

Откройте: **http://localhost:8000**

Или если используете Apache: **http://localhost/DBGranit**

## Решение проблем

### "Ошибка подключения к базе данных"
- Убедитесь, что MySQL запущен
- Проверьте настройки в `config.php`
- Убедитесь, что база данных `ритуальные_услуги` создана
- Используйте веб-интерфейс `setup_database.php` для создания БД

### "Порт 8000 занят"
- Используйте другой порт: `C:\xampp\php\php.exe -S localhost:8080`
- Или закройте программу, которая использует порт 8000

### "Страница не найдена"
- Убедитесь, что вы находитесь в правильной папке при запуске сервера
- Проверьте, что файл `index.php` существует в корне проекта
