    ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРИЛОЖЕНИЯ
================================

Для запуска приложения необходимо:

1. УСТАНОВИТЬ PHP
   - Скачайте PHP с официального сайта: https://www.php.net/downloads.php
   - Или установите XAMPP (включает PHP, MySQL, Apache): https://www.apachefriends.org/
   - Или установите OpenServer: https://ospanel.io/

2. УСТАНОВИТЬ И ЗАПУСТИТЬ MYSQL
   - Если используете XAMPP: запустите MySQL через панель управления XAMPP
   - Если используете OpenServer: MySQL запускается автоматически
   - Если MySQL установлен отдельно: убедитесь, что сервис запущен

3. СОЗДАТЬ БАЗУ ДАННЫХ
   - Откройте phpMyAdmin (если используете XAMPP/OpenServer)
   - Или используйте MySQL Workbench, или командную строку MySQL
   - Выполните SQL скрипт из файла database.sql

4. НАСТРОИТЬ ПОДКЛЮЧЕНИЕ
   - Откройте файл config.php
   - Измените параметры подключения к БД при необходимости:
     * DB_HOST - адрес сервера (обычно localhost)
     * DB_USER - пользователь MySQL (обычно root)
     * DB_PASS - пароль MySQL (обычно пустой для локальной разработки)
     * DB_NAME - имя базы данных (ритуальные_услуги)

5. ЗАПУСТИТЬ СЕРВЕР

   Вариант А: Через командную строку
   - Откройте командную строку в папке проекта
   - Выполните: php -S localhost:8000
   - Откройте браузер: http://localhost:8000

   Вариант Б: Через XAMPP
   - Скопируйте папку проекта в htdocs (обычно C:\xampp\htdocs\)
   - Запустите Apache через панель XAMPP
   - Откройте: http://localhost/название_папки

   Вариант В: Через OpenServer
   - Скопируйте папку проекта в domains
   - Выберите домен в OpenServer
   - Откройте выбранный домен в браузере

   Вариант Г: Через батник (если PHP в PATH)
   - Дважды кликните на start.bat

6. ОТКРЫТЬ В БРАУЗЕРЕ
   - Перейдите по адресу: http://localhost:8000
   - Или используйте адрес, указанный вашим веб-сервером

ПРОБЛЕМЫ И РЕШЕНИЯ:
- "PHP не найден" - добавьте PHP в переменную PATH или используйте XAMPP/OpenServer
- "Ошибка подключения к БД" - проверьте, что MySQL запущен и настройки в config.php правильные
- "База данных не найдена" - выполните SQL скрипт database.sql для создания БД
